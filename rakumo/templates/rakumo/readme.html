メンバー周り(グループとその所属)の情報取得、編集

  • グループリスト取得
    グループ(メーリス)の一覧を取得します。「CSVダウンロード」ボタン押下でCSVで出力します。 - kind：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - id：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - etag：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - email：グループのメールアドレスです。
    - name：表示用のグループ名です。
    - directMembersCount：グループに登録しているメンバーの数を表します。
    - description：グループを説明する詳細文です。
    - adminCreated：：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - nonEditableAliases：自動付与でのメールエイリアスです。
    - aliases：変更可能なメールエイリアスです。
  • グループ毎所属 メンバーリスト取得
    グループに所属するメンバー全体の情報を一覧で取得します。「CSVダウンロード」ボタン押下でCSVで出力します。
    - groupId：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - groupEmail：グループのメールアドレスです。
    - groupName：表示用のグループ名です。
    - groupMemberCnt：グループに登録しているメンバーの数を表します。
    - memberId：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - memberEmail：メンバーのメールアドレスです。
    - memberRole：メンバーのグループ所属内での権限を表します。
    - memberType：メンバーのタイプを表します(USER・・・ユーザー、GROUP・・・下層になっているグループ、CUSTOMER・・・外部アドレス)
    - memberStatus：メンバーのグループ内でのステータスを表します。(ACTIVE・・・利用中、SUSPENDED・・・利用停止)
  • グループメンバー追加
    グループ(メーリス)に所属するメンバーを追加します。テンプレートを利用して実行してください。
  • グループメンバー削除
    グループから対象メンバーを削除(離脱)します。テンプレートを利用して実行してください。
  • グループメンバー更新 テンプレート取得
    グループメンバーの情報を追加、削除するテンプレートを取得します。
    - groupKey：所属するグループのメールアドレス
    - memberKey：登録するメンバーのメールアドレス
  • メンバーリスト取得
    G-Suiteに登録しているメンバー(メールアドレス)の一覧を取得します。
    - primaryEmail：G-Suiteアカウントとなるメンバーのメールアドレス
    - fullName：表示用のメンバー名です。

権限周り(カレンダーの閲覧、編集(代理入力など)権限)の情報取得、編集

  • 権限リスト取得
    カレンダーに登録しているメンバーへのアクセス権限をメンバー毎に情報を一覧で取得します。 - calendarId：対象メンバーのメールアドレス
    - kind：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - etag：G-Suite用システム用の値です。(特に気にしなくて良いです。)
    - id：権限用のIDです。削除時に使います。
    - scope_type：対象のメンバーに対するアカウント(ユーザ、グループ)のタイプです。(user・・・ユーザー、group・・・グループ)
    - scope_value：対象のメンバーに対する権限を持つアカウントのメールアドレスです。
    - role：対象のメンバーに対する権限を持つアカウントの権限タイプです。(reader・・・閲覧権限(すべての予定の詳細)、writer・・・予定の変更権限、owner・・・変更及び管理の共有権限、freeBusyReader・・・予定の時間枠のみ表示(詳細を非表示))
  • 権限追加、更新
    カレンダーに登録しているメンバーへの権限を追加します。すでに登録している場合は権限のみ変更します。(登録するとお知らせメールが送信されます。)
  • 権限削除
    カレンダーに登録しているメンバーへの権限を削除します。

施設周り(会議室、備品、車など)の情報取得、更新

  • 施設リスト取得
    G-Suiteに登録している施設情報の一覧を取得します。
  • 施設情報更新
  • 施設情報更新 テンプレート取得
